# CommitList

CommitList is a frontend-only smart task board (static HTML + Tailwind CSS) with demo data for users, tasks, tags, and task details.

## Local setup

```bash
npm install
npm run build
```

Then open `dist/index.html` in your browser.

## Deploy (Vercel)

This repo is configured to build using `npm run build` and deploy the `dist/` folder.
